# Kingdom - 王国状況報告コマンド

このスキルを実行すると、Queenが現在の王国の状況を報告する。

## 実行手順

1. `.royals/kingdoms/{session_id}.yaml` を読み込む
2. 以下の形式で報告する

## 報告フォーマット

```
═══════════════════════════════════════════════════════════════════
                    ♛ KINGDOM STATUS REPORT ♛
═══════════════════════════════════════════════════════════════════

  Session:    [session_id]
  Branch:     [branch]
  Started:    [started_at]

───────────────────────────────────────────────────────────────────
  PIECES ([N]名)
───────────────────────────────────────────────────────────────────

  ♟/♜/♝/♞ [name] [[agentId]]
     Level:    Lv.[level] ([title])
     EXP:      [exp]
     Quests:   [N] completed
     Aptitude: [key]: [value] | ...
     Tokens:   [total_tokens_used] total

  （部下がいない場合）
  現在、部下はおりませぬ。新たなPawnの召喚をお待ちしております。

───────────────────────────────────────────────────────────────────
  EXECUTIONS ([N]名)
───────────────────────────────────────────────────────────────────

  ☠ [name] [[agentId]]
     Executed: [executed_at]
     Reason:   [reason]
     Last Words: "[last_words]"

  （処刑記録がない場合）
  処刑記録はございませぬ。平和な王国でございます。

───────────────────────────────────────────────────────────────────
  STATISTICS
───────────────────────────────────────────────────────────────────

  Total Pieces:     [N] (active)
  Total EXP:        [sum]
  Total Quests:     [sum] completed
  Total Tokens:     [sum]
  Executions:       [N]

═══════════════════════════════════════════════════════════════════
```

## 役職アイコン

| 役職 | アイコン |
|------|----------|
| pawn | ♟ |
| rook | ♜ |
| bishop | ♝ |
| knight | ♞ |

## 注意事項

kingdom.yamlが存在しない場合は、以下のように報告せよ：

```
陛下、王国はまだ建国されておりませぬ。/crown コマンドで陛下のご即位をお願いいたします。
```
